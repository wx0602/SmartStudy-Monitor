# SmartStudy Monitor 🧠

基于 Python + MediaPipe的智能学习监督系统

SmartStudy Monitor 是一款桌面端智能学习辅助工具，结合计算机视觉与行为分析，可实时监测专注状态、不良行为与坐姿，并提供提醒与时间管理功能，帮助提升学习效率。

### ✨ 功能特色

-🚀 **AI 行为监测**

检测：玩手机、离席、托腮、摸脸

基于 YOLOv8 + MediaPipe + 时序过滤，降低误判

语音提醒 + 弹窗反馈

-👀 **专注度与疲劳分析**

FaceMesh 头部姿态与视线追踪

EAR 眼部疲劳检测（嗜睡 / 长时间闭眼）

个性化 基线校准，输出 0–100 专注评分

-🧍 **坐姿与体态评估**

驼背、高低肩、颈部前伸检测

屏幕距离与坐姿端正度评估

MediaPipe Pose + 尺度归一化

-📊 **实时数据面板**

专注度 / 疲劳值 / 分心值

行为状态颜色提示 + 进度条可视化

-⏰ **时间管理工具**

当前时间与日期显示

倒计时 / 计时器 / 闹钟

待办清单（任务持久化）

-⚙️ **系统设置**

行为检测独立开关

音量调节与参数保存

深色 / 浅色主题切换

-🖥 **技术栈**

Python 3.9.13

PyQt5（GUI）

OpenCV（视频处理）

MediaPipe（人脸 / 姿态 / 手部）

YOLOv8（手机检测）

-💻 **运行环境**

Windows 10 / 11

需要摄像头设备

### 快速开始
1. 安装依赖：`pip install -r requirements.txt`
2. 运行程序：`python main.py`
